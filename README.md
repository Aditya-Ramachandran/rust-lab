# 🐞 rust-lab

A collection of Rust experiments, exercises, and mini-projects built while learning the language and exploring systems programming.

The goal of this repository is not to complete tutorials, but to develop a solid understanding of Rust by building things: command-line tools, networking applications, concurrent systems, and infrastructure-inspired projects.

Ultimately, this serves as the foundation for larger projects such as **Hermes**, a lightweight Kafka-inspired message broker written in Rust.

---

## Why This Repository Exists

As a backend engineer coming from a Java/Spring background, I wanted to learn Rust through practical projects rather than isolated examples.

This repository documents that journey:

* Understanding ownership and borrowing
* Learning Rust's concurrency model
* Building async applications with Tokio
* Exploring file systems, networking, and persistence
* Developing systems programming skills
